# Saaty's Multi-Level Assessment

Веб-приложение на `Streamlit` для многоуровневой оценки факторов по методу Саати (МАИ, AHP).

## Что делает приложение

- собирает попарные сравнения для главных критериев (уровень 1);
- собирает попарные сравнения для подфакторов внутри каждой группы (уровень 2);
- проверяет согласованность матриц (ОС / CR);
- рассчитывает локальные и глобальные веса факторов;
- формирует итоговую таблицу рейтинга факторов;
- после успешной проверки формирует CSV с ответами и весами.

## Структура проекта

- `app.py` — основное приложение Streamlit;
- `requirements.txt` — зависимости проекта;
- `saaty_answers_and_weights.csv` — пример экспортированного файла.

## Установка и запуск

1. Клонируйте репозиторий:
   - `git clone https://github.com/KTL-deep/Saaty-s-Multi-Level-Assessment.git`
   - `cd Saaty-s-Multi-Level-Assessment`
2. Установите зависимости:
   - `python -m pip install -r requirements.txt`
3. Запустите приложение:
   - `streamlit run app.py`
4. Откройте в браузере:
   - `http://localhost:8501`

## Как пользоваться

1. На шаге 1 задайте попарные сравнения главных критериев.
2. На шаге 2 заполните сравнения подфакторов в каждой вкладке.
3. Нажмите кнопку **"Проверить согласованность и рассчитать результат"**.
4. Если все матрицы согласованы, приложение:
   - покажет итоговый рейтинг факторов;
   - выведет сумму глобальных весов;
   - покажет кнопку **"Скачать CSV с ответами и весами"**.

## Формат CSV

Экспортируемый файл содержит строки двух типов:

- `Ответ` — пользовательские попарные оценки:
  - `Группа`, `Фактор A`, `Фактор B`, `Выбор`, `Оценка Саати`, `Значение A/B`;
- `Вес_группы` и `Вес_подфактора` — рассчитанные веса:
  - `Локальный вес`, `Глобальный вес`.

Файл отдается в кодировке `utf-8-sig`, поэтому корректно открывается в Excel с кириллицей.
